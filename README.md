# Library Management System

A Django-based web application for managing books and user reading activity.

## Features
- User registration & login
- Personalized dashboard
- Book management (add, delete, view)
- Reading insights
- Clean modern UI

## Technologies
- Django (Python)
- HTML, CSS
- SQLite

## Run locally
```bash
git clone https://github.com/SaraSahitaj/library-management-system.git
cd library-management-system
pip install -r requirements.txt
python manage.py runserver
